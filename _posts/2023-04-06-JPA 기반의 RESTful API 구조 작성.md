Pastry 프로젝트에서 사용할 기술 스택은 일단 아래와 같이 정했다.

1. JPA를 이용하여 데이터베이스와의 연동을 구현하기로 하자. 

2. 데이터베이스는 MySQL을 사용하여 데이터를 저장하고 관리하도록 하자.

3. Java 언어와 Spring Boot 프레임워크를 사용하여 서버 개발을 하자.

4. RESTful API를 사용하여 클라이언트와 서버 간의 통신을 구현하자.
